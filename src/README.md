# Trabalho de Modelagem 3D

## Jogo na Unreal Engine

O trabalho foi feito na Unreal Engine e foi usado alguns assets do marketplace da Epic Games, é um simples jogo que você precisa subir alguns objetos para ver uma mensagem te parabenizando por ter conseguido subir lá, foi usado diferentes texturas, luzes dinâmicas, objetos do marketplace da Epic Games e uma textura feita para fazer a mensagem de parabenização e para colocar os créditos para mim mesmo por ter feito o mapa inteiro.
